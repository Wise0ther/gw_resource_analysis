# 📊 GWars - Анализ Торговли Ресурсами

Пользовательский скрипт для игры **GWars.io**, который по запросу собирает историю передач и рассчитывает чистую прибыль от торговли ресурсами с учетом сопутствующих расходов (лицензия техника, аренда транспорта).

---

### 🚀 Быстрая установка

1. Установите расширение для браузера **Tampermonkey**.
	- Если используете браузер **Mozilla Firefox** нажмите на ссылку **[Tampermonkey в Firefox Add-ons](https://addons.mozilla.org/ru/firefox/addon/tampermonkey/)**
	- Если используете браузер **Google Chrome** нажмите на ссылку **[Tampermonkey в  интернет-магазин chrome](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=ru)**
2. Нажмите на кнопку ниже для установки скрипта:

👉 **[УСТАНОВИТЬ СКРИПТ](https://github.com/Wise0ther/gw_resource_analysis/raw/main/gwars-res-calc.user.js)** *(откроется окно Tampermonkey с предложением установить)*.

---

### ✨ Основные возможности

* **Сбор логов за заданный период:** На странице транзакций.
* **Учет расходов:**
  * Учет стоимости лицензий (1, 7, 30 дней). Учитывает только сумму потраченную на покупку лицензии.
  * Учет стоимости аренды транспорта.
* **Детализация по ресурсам:** Показывает объемы продаж/покупок, средние цены за единицу и итоговый профит по каждому ресурсу отдельно.
* **Изоляция данных:** Хранит логи и настройки отдельно для каждого `user_id`.

---

### 🛠 Инструкция по использованию

1. Зайдите на страницу **Протокол передач** >> **Проткол транзакций** (`https://www.gwars.io/transfers.php?user_id=*`) вашего персонажа.
2. В правом верхнем углу рабочей области появится кнопка **«📊 Калькулятор Ресурсов»**.
3. Задайте промежуток дат. Нажмите **ОК**
4. Укажите накладные расходы (`лицензия и аренда транспорта`)
5. Нажмите кнопку **«Обновить данные»**, чтобы скрипт собрал историю передач.
6. После сбора данных получите сводку.
